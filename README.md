# Customer Sales ETL Pipeline

## Overview

This project demonstrates a simple ETL pipeline using Python and SQLite.

## Technologies

- Python
- Pandas
- SQLite

## ETL Process

### Extract
Read sales data from CSV.

### Transform
- Remove duplicates
- Calculate total sales amount

### Load
Store transformed data in SQLite database.

### Reporting
Generate customer spending report.
